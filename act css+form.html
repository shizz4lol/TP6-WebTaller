Respuestas actividad:

CSS significa "Cascading Style Sheets" o sea hojas de estilo en cascada, y es un programa utilizado para darle el diseño a los documentos HTML
La diferencia entre aplicar estilos en linea, en y como archivo externo seria basicamente la comodidad y segun cuantos elementos necesiten un estilo. Si necesito que solo un elemento tenga cierto diseño, puedo usar css en linea. Ahora, si voy aplicar el mismo diseño a varias paginas, un archivo externo es lo mejor, ya que se puede vincular con tantos archivos HTML como sea necesario.

Selectores de etiqueta

Permiten aplicar el mismo estilo a un tipo de etiqueta en especifico (p, h1, h3, etcetera)

Clase

Conjunto de propiedades/estilos que son aplicados a distintos elementos. Por ejemplo ".libros"

ID

Tambien es un conjuto de propiedades, pero se aplican solo a un elemento en especifico porque el ID es unico en todo un HTML. Por ejemplo "#dni".
